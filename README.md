# Alpha Lab — 美股入场信号分析

**Live site → https://vambrocop.github.io/alpha-lab/**

用统计学 + 贝叶斯概率分析美股（NASDAQ）的历史规律，给出每日入场信号参考。

## 功能

- **贝叶斯综合信号** — 把月份胜率、星期效应、月内周次、假日效应、税季、技术指标（MA/RSI/BTC动量）叠加成一个概率
- **未来30日前瞻** — 扫描未来45个交易日，预测哪几天日历因子最强/最弱
- **事件研究（Event Study）** — 用1928-2026历史数据，实证计算加息/降息/贸易战/地缘冲击等事件后30日的真实超额收益
- **多元统计分析** — CCA、SHAP因子重要性、Prophet预测、卡尔曼趋势、路径分析（SEM）
- **长历史统计（1928+）** — 百年视角下的月度/星期/假日效应

## 数据来源

- Yahoo Finance（NASDAQ / S&P 500 / VIX / DXY / BTC / 黄金 / 原油 / 半导体指数）
- FRED（美联储利率 / M2 / CPI / 失业率）
