# 持仓感知告警 —— 持仓清单模板
#
# 用途：只在你【实际持有】的票出现风险事件（距 52 周高深度回撤）时，
#       才单独 Telegram/Issue 提醒你；不是买卖建议，只是风险抬头。
#
# 怎么用（二选一）：
#   A. 本地测试：把本文件复制为同目录下的 holdings.txt（已被 .gitignore，不会进公开仓库），
#      每行写一个美股代码。然后跑：
#         $env:HOLDINGS=''; py market-analysis/scripts/alert_check.py
#   B. 线上（GitHub Actions 自动推送）：在仓库 Settings → Secrets 里加一个名为 HOLDINGS 的 Secret，
#      值为逗号或换行分隔的代码，例如：NVDA,MSFT,TTD
#      （放 Secret 而不是提交文件 = 你的持仓不会公开。）
#
# 说明：
#   - 用美股代码，且与本站口径一致（如伯克希尔写 BRK-B 不是 BRK.B）。
#   - 目前只覆盖被追踪的 Valpha150 成分股；不在其中的任意票暂不告警（可用「点单深算」单查）。
#   - 触发条件是“距 52 周高的回撤”跨档：进入 >25%（深度）/ >40%（重度）各提醒一次，收窄回正常再提醒一次。
#
