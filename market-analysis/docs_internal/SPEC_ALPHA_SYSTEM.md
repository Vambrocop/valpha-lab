## 🧠 人脑轨（精选 + 工具；公开、诚实框架）= v1.5

### Layer ① Valpha150 大盘（"我们自己的指数"）
- ~150 只**精选**（非机械全包）。来源池：NASDAQ-100 龙头 + 费城半导体(SOX) + S&P 非科技大蓝筹 + 主脑挑的热点。
- **入池标准**（curate，不硬塞）：市值/流动性 + 板块代表性（每板块留龙头）+ 主题/动量 + 风险体检有看点。
- **不强制全包 S&P100/SOX**：有代表性的进，没看点的不塞。
- **定期复审**：季度复盘 + changelog（加/删/理由/日期，append-only）；持续表现的轮进 20，掉队轮出。
- **展示**：可排序大表（点列头切：板块 / 涨幅 6月·1年·10年 / 异动 / 主脑分 / 风险）+ 板块**色标** + 主脑标签(🔥动量 🏰基石 ⚠️高风险 💎看好) + 顶部板块占比条 + 筛选。
- **诚实标注**：涨幅榜 = 幸存者偏差 + 追涨风险；主脑推荐 = 观点非保证。
- **指数追踪**：S&P 500(^GSPC) + NASDAQ + **费城半导体(SOX)** 一起；每个标"怎么买"(ETF：SPY/VOO、QQQ、SOXX/SMH)。

### Layer ② 20 重点观察（深度体检 + 雷达评分）
- 14 常任(蓝筹基线) + 6 非常任(动量轮换·跨主题：闪迪/美光/超微/帕兰提尔/Coinbase/优步)。
- 每只**多维雷达图**(用现成模型/指标，export_stocks 已算)：动量 / 趋势 / 风险(低=好) / 估值性价比 / 系统性(β) / 超跌反弹。各维 0-100。
- 是 Layer③ 的选股池。

### Layer ③ 配置建议器（出格区·个人化）
- 输入：金额(**澳元**) + 持有期(短/1年/长) + 选股(20 里 / 手输任意 / **我推荐**)。
- 输出：风险加权配置 + 持有期 + 怎么买(一次性 vs 分批) + **诚实账单**(组合历史最深回撤/最差一年/波动) + **诚实回测**(过去 N 年路径含最深坑) + **追涨警告**(选到涨幅榜顶时)。
- 逻辑：逆波动率 + 相关性去重 + 持有期调；用 Layer② 雷达/体检数据。
- 架构：前端交互(像现有澳元/CGT 计算器)，浏览器算，无后端。
- 🔴 重免责：教育非建议·会亏·过去≠未来；**新手优先、大白话、不怂恿集中押注**。
