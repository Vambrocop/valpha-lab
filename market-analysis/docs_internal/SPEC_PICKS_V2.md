## 0. 诚实定位(B1·两层真相都说)
- **规则层(有先验)**:v2 唯一改动 = 动量窗改为**跳月式:取 t-147→t-21 收盘(126 交易日窗,
  跳过最近 21 个交易日)**——跳过最近一月规避短期反转是 Jegadeesh-Titman(1993)以来动量文献的
  标准构造。命名 **"6-1 跳月动量"**(126 日≈6 个月;不叫"12-1"——那是 12 个月回看的文献术语,误称)。
- **采纳层(事件驱动·必须公开披露,进 caveat 与 PICK_RULE_V2)**:
  > "v2 上线动机 = v1 首批(2026-06-26)翻车后的改版;规则选择虽有文献先验,**采纳的时点与
  > '选哪个改良'本身受该次亏损事件驱动**——v2 战绩需较长横期,且只能算'看过一次亏损后的
  > 一个改良',别当干净的事前 OOS 证据。"
- **除动量窗外零改动**:lowvol63、等权、头尾各 3、HOLD_TD=20、BENCH=QQQ、同一观察池全同 v1。
- **禁止**:额外因子/阈值/过滤(DoF);历史回测挑参(19 只幸存者池回测无意义,直接前向计分)。
- **v1 照跑照认账**;非投资建议、会错、过去≠未来。

## 1. 实现
### 1.1 选票函数(B2:两处耦合改动,不是"一行")
`pick_ledger.py` 加 `_select_picks_v2(prices)`,与 `_select_picks` 差异**恰两处**:
- 动量:`mom = px.iloc[-1-21] / px.iloc[-1-21-126] - 1`(= iloc[-22]/iloc[-148];t-21 收盘/t-147 收盘);
- 守门:`if len(px) < MOM_WIN + 21 + 1: return []`(**148 行**——沿用 v1 的 127 会 IndexError)。
其余(vol/rank/head-tail)逐行同 v1。**不改 `_select_picks` 本体**(v1 与 AU 在用)。

### 1.2 编排(S2:run() 本体一字不改)
- **新增独立 `run_v2()`**;`__main__` 编排 `run()` → `run_v2()`(v1 先写完,v2 后跑)。
- **B3·v2 整腿 fail-soft**:run_v2 的取价+选票+结算+json 注入**全程 try/except**,任何异常打印跳过、
  绝不 raise 出 `__main__`(run_all 对非零退出会中止全流水线+封存+发布门——v2 崩不许连坐 v1)。
- **picks.json 合并机制(写死)**:run() 照旧写 v1 版 → run_v2 读回、置 `out["v2"] = {...}`、重写。
  v2 块含:PICK_RULE_V2/track_record/recent/verdict/上线日期。
- **S3·各取各的价**:run_v2 对**自己账本的 unsettled 符号**独立 `fl.fetch_prices`,不共享 v1 的
  prices 字典(防 v2 挂账票不在 v1 集合被误判 dropped)。两账本各读各的 LOG、各建 seen,零交叉。
- 独立账本 `data/pick_ledger_v2.csv`(HEADER 同 15 列;append-only)。
- **S4·sidecar**:`ledger_sidecar.py` SPECS 追加 `("pick_ledger_v2.csv", ["pick_date","symbol","view","mom_pct"])`。

## 2. 测试(≥9)
1. 分歧机制:合成"最近 21 日暴涨票"——v1 选中、v2 不选;
2. 窗口精确:v2 动量 == `iloc[-22]/iloc[-148]`(合成面板 **≥148 行**·off-by-one 守门);
3. 两账本互不污染;4. v2 幂等 append-only;
5. **v1 回归靶子(S2 澄清)**:= `pick_ledger.csv` 逐字节 + `run()` 返回 dict 不变
   (**不是** picks.json 整文件——它按设计多 v2 块);
6. 短面板守门:128~147 行 → v2 返回 [] 不抛(B2);
7. sidecar:pick_ledger_v2.csv 可 seal+verify(S4);
8. 合并完整性:picks.json 同时有完整 v1 字段与 v2 块、v1 子结构未被破坏;
9. v2 fail-soft:v2 腿抛异常 → v1 输出与退出码不受影响(B3);
10. (N3)最近 21 日走平的合成面板 → v1 选票 == v2 选票(重合 sanity,示范 S5)。
