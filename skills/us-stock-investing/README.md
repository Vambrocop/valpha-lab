# US Stock Investing Skill

A comprehensive Chinese-language investment analysis framework for beginner US stock investors, covering:

- Valuation metrics (P/E, P/S, PEG, EV/EBITDA) explained in plain language
- Stock ratings for 2026 across 4 categories (conservative / growth / AI infrastructure / speculative)
- Monthly seasonality data (win rates for all 12 months, 1950鈥?024)
- Presidential election cycle patterns
- AUD investor-specific guidance (platforms, tax, FX)
- AUD$6,000 three-tranche strategy with and without NVDA
- Macro risk radar (CAPE, Fed rate, tariff risk)
- 8-question buy checklist

## Usage

Place in your Claude Code skills directory:
```
~/.claude/skills/us-stock-investing/main.md
```

Or reference directly in your analysis workflow.
